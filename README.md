###### SENSORES Y NODO
	Lenguaje de programación:
   		- C++ para Arduino  
	Librerías:
    		- Arduino Json versión 5.0: Librería para poder hacer funcionar la librería de FirebaseArduino
    		- ESP8266WiFi: Librería para poder conectar el NodeMCU a la red WiFi
